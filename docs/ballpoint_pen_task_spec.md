# Ballpoint Pen Collection — 任务规格书

> 任务名称: Ballpoint Pen Collection（圆珠笔收纳）
> 统一 Prompt: `Grab ballpoint pen and drop into pen holder.`

> 文档版本: 2026-04-18

---

## 1. 目标物体（Target）

| 属性 | 值 |
|------|-----|
| 物体 | 圆珠笔 ×4 |
| 候选池 | 3 黑 + 2 红 + 1 蓝 = 6 支（笔帽颜色区分，笔身一致） |
| 抽取方式 | 每条 episode 从 6 支中随机取 4 支 |
| 颜色约束 | 不强控组合，自然多样性即可 |

## 2. 干扰物（Distractor）— 固定不变

> 干扰物总数: 6 件（2 橡皮 + 3 其他笔 + 1 笔芯盒）
> **每条 episode 干扰物配置完全相同，仅位置可随机。**

## 3. 容器

pen holder（笔筒）×1，固定款式，位置随机（4 选 1）。**不允许使用其他容器替代。**

## 4. 笔筒位置（4 选 1）

每个 episode 开始前，笔筒放置于以下 4 个固定位置之一。以桌面自然纹理为参考定位，无需贴胶带。

### 桌面坐标系定义

```
        后方（远离操作者）
     ┌─────────────────────┐
     │   P3          P4    │
     │                     │
     │                     │
     │   P1          P2    │
     └─────────────────────┘
        前方（靠近操作者）
          ← 机械臂基座
```

## 5. 初始分布模式 — 随机分布

## 6. 机械臂规范

起始和最终位置一样

### 6.2 Episode 流程

```
1. 机械臂移动到 home pose
2. 操作者按随机分布协议布置场景（笔筒位置 + 物体散布）
3. 确认布置完毕 → 开始录制
4. 遥操作: 逐个抓取 4 支圆珠笔投入笔筒
5. 机械臂回到 home pose → 结束录制
```

### 6.3 时间限制

| 参数 | 值 |
|------|-----|
| 最大时长 | 25 秒 |
| 帧率 | 30 FPS |

## 7. 采集计划

### 7.1 数据量

| 阶段 | Episode 数 | 说明 |
|------|-----------|------|
| 初始采集 | 50 | 用于首次训练验证 |
| 正式采集 | 100–150 | 模型收敛所需量级 |

### 7.2 质量检查

* [ ] 每个 episode 确认 4 支笔全部进入笔筒
* [ ] 无碰撞、掉落等异常
* [ ] 时长 ≤ 20 秒
* [ ] prompt 文本完全一致
* [ ] 干扰物配置与照片一致
